# Django Complaint Management System

This is a Django-based web application for managing complaints within an organization. The application allows users to register complaints, track their status, respond to complaints, and escalate them when necessary.

## Features

- User registration and authentication
- Complain registration with heading and description
- Assigning complaints to designated authorities within the organization
- Tracking the response time and escalation of complaints
- Allowing designated authorities to respond to and close complaints
- Like functionality for users to express their agreement or support for a complaint
- Offensive word detection to prevent inappropriate content

5. Start the development server:

```python manage.py runserver```

6. Open your web browser and visit [http://localhost:8000](http://localhost:8000) to access the application.

## Usage

1. Create a user account or log in with your existing credentials.
2. Register a new complaint by providing a heading and description.
3. Designate the complaint to the appropriate authority within the organization.
4. Track the status of your registered complaints on the dashboard.
5. Designated authorities can respond to and close the assigned complaints.
6. Users can like or unlike a complaint to show their support or disagreement.
7. Escalate a complaint if necessary, which will be assigned to a higher authority.
8. Users can reopen a closed complaint if they are not satisfied with the resolution.
