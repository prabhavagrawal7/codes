\section*{Title of the Project}
Grievance Redressal System
\section*{Name of the Supervisor(s)}
Dr. Niharika Anand
\section*{Abstract}
In this project, we will develop a web-based grievance redressal system using Django, various data structures including trees and graphs, CSS, JavaScript, and Bootstrap. The system will be designed to handle and track complaints or reports lodged by faculty and students. A smart system for addressing and escalating complaints will also be implemented using a tree and graph data structure, which will include automatic escalation if a complaint is not resolved within a predetermined time. This system will provide the facility to the users to see the hierarchical representation of the complaints and the authority to which it is escalated. Additionally, a user login system will be added to allow users to track the status of their complaints and receive updates.

\section*{Overview of Research Problem/Project}
The project aims to develop a web-based grievance redressal system to address the problem of inefficient and ineffective grievance redressal in a college or university setting. The system will allow students and faculty to lodge complaints in writing, track the status of complaints, and include automatic escalation to ensure timely resolution. The system will be built using Django, data structures including trees and graphs, CSS, JavaScript, and Bootstrap. It will also provide a hierarchical representation of complaints and authorities and a user login system for tracking and updates.
